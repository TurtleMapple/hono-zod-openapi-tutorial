# Hono Zod OpenAPI Tutorial
Tutorial implementasi REST API menggunakan Hono, Zod, dan OpenAPI dengan dokumentasi Scalar UI.

## Features
- ✅ Type-safe API dengan Zod validation
- ✅ Auto-generated OpenAPI documentation
- ✅ Interactive API documentation dengan Scalar UI
- ✅ TypeScript support

## Quick Start
```bash
# Clone repository
git clone https://github.com/[username]/hono-zod-openapi-tutorial.git
cd hono-zod-openapi-tutorial

# Install dependencies
pnpm install

# Run development server
pnpm dev
